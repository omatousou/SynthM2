# Plot de Courbe

Ce script Python génère un plot d'une courbe sinusoïdale en utilisant matplotlib et numpy.

## Installation des dépendances

Installez les packages requis avec pip :

```
pip install -r requirements.txt
```

## Exécution

Lancez le script :

```
python plot_curve.py
```

Cela ouvrira une fenêtre avec le plot de la courbe sin(x) de 0 à 10.

## Dépannage

- Assurez-vous que matplotlib et numpy sont installés.
- Si le plot ne s'affiche pas, vérifiez que votre environnement supporte l'affichage graphique (par exemple, sur Windows, Tkinter doit être disponible).
